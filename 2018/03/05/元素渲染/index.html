更新元素渲染

React 元素都是immutable不可变的。当元素被创建之后，你是无法改变其内容或属性的。一个元素就好像是动画里的一帧，它代表应用界面在某一时间点的样子。

这个例子里面 通过 setInterval() 方法，每秒钟调用一次 ReactDOM.render().更新渲染页面。
