弄清原理之React Router

1. 概念

在web应用开发过程中，路由系统是不可或缺的一部分，路由简单来说就是当url发生变化时，web界面也会随之改变。react 也有自己的路由系统，那就是 react router 路由系统。

2. 基本原理

url 和ui 是意义对应的关系，记当你输入Url 时，就回渲染对应 react component
