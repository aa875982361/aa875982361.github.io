动态规划

Posted on 2018-07-08

今天在团队群里面 有人分享了一遍文章，描述动态规划，感觉挺有趣的，但是看不懂，所以自己上网找了一下资料

先给出我参考的一篇文章： 漫画：什么是动态规划？

我比较习惯通过一个具体的例子去学习，所以我们来看个小例子吧！

题目：

有一座高度是10级台阶的楼梯，从下往上走，每跨一步只能向上1级或者2级台阶。要求用程序来求出一共有多少种走法。

怎样才能到达第10阶呢？

这就有两种情况了，因为有可以走一步 或者可以走两步

  1. 从第9阶 走一步
  2. 从第8阶 走两步

我们现在暂且不知道 走到第8阶，第 9阶有多少种走法，但是我们可以设置一个 变量去表示，就像 :

  1. 走到第8阶有 X 种 走法
  2. 走到第9阶有 Y 种 走法

  3. 走到第10阶有 Z 种 走法

    那么就会有 Z = X + Y 这样的关系

    那 在仔细想一下 走到第9阶 有多少种走法？

    走到 第9 阶 == 走到 第8阶 + 走到第7阶

    所以是不是就得到这样的关系

    F（10） = F（9）+ F（8）；
    F（9）= F（8）+F（7）；
    ….
    ….
    ….
    所以是不是就得到这样的关系：

    F(N) = F(N-1) + F(N-2) (N>=3)

    那要到哪里才停止呢？

    再看看题目：

    每跨一步只能向上1级或者2级台阶

也就是说

  1. 走到第1阶 有一种走法，就只走1步
  2. 走到第2阶 有两种走法，从第1阶到第 2 阶，从一开始 直接到第 2 阶
  3. 走到第三步 就是从第2阶 到第3阶 或者是 从第1阶走2步 到第3阶
  4. 。。。。
