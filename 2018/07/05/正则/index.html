正则

Posted on 2018-07-05

以前一直没有仔细看过正则的用法，之前都是直接用别人的正则，趁现在有空，学习一下

ES通过RegExp 类型来支持正则表达式，使用下面类似的Perl 的语法，就可以创建一个正则表达式。

var expression = /pattern/flag;

其中模式（pattern) 部分可以是任何简单或负责为的整个表达式，可以包含字符类、限定符、分组、向前查找以及反向引用。每个正则表达式都可带有一个或多个标志（flag），用来标明正则表达式的行为，正则表达式的匹配模式支持下列3个标志。

  • g： 表示全局（global）模式，这种模式被应用于符合条件的全部字符串。
  • i ：表示不区分大小写（case-insensitive）模式，即在确定匹配项时，立即停止。
  • m：表示多行（multiline）模式，即在达到一行文本末尾之后还会继续寻找下一行匹配的项

因此一个正则表达式就是一个模式与上面3个标志的结合体。

使用正则表达式字面量和使用RegExp构造函数创建的正则表达式不一样，
在es3 中，正则表达式字面量是始终共享同一个实例，而构造函数创建的每一个新的RegExp都是一个新的实例.
但是在es5之后 使用正则表达式都必须像直接调用RegExp 构造函数一样，每次创建一个新的实例。

可以看到在现在的浏览器上面 输出都是10个true 。

RegExp 属性

属性名 类型 说明
global Boolean 是否设置了 g 标志
ignoreCase Boolean 是否设置了 i 标志
lastIndex Number 表示考试搜索下一个匹配项的字符位置，从0算起。
multiline Boolean 是否设置了m 标志
source string 正则表达式的字符串表示，按照字面量表示

  • 输出 数组 （Array）： 但是包含两个二外属性 ：index 和 input
